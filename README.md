<h2>Beauty Parlor Management System</h2>
In this project i  have created a web-application for beauty parlor management 
<br>
This Web-application is help to Book an Appointment, online payment, view book appointment and also check the history of appointments and help to manage all the appointment
<br>
This project is created to reduce the man work and make booking easy.
<br>
The Tech we use to build this project are - 
<br>
  1.Frontend - HTML, CSS, JS
  <br>
  2.Backend -Python
  <br>
  3.Framework - FLASK(Python)
  <br>
  4.Database - MySQL  
<br>
<h3>Mandatory Requirements to run this whole Project by using Visual Studio Code</h3>
1. Python Should be install in your PC/Laptop
<br>
2.For This Flask should be install to run the program and handle the flask libraries.
<h3>Details Of Project</h3>
This Project is divied into two type 
<br>
 1.User
 <br>
 2.Admin
<h4>User</h4>
In the <u>User Side</u> They can register themself by login page and if they  already have so they can login directly
<br>
Then in the Home page there is a nav bar where multiple option are present like - Service, Product, Appointment, AboutUs, and Contact
<br>
In the Home page user get to know about the varity of services we provide and the product is use in the parlor
<br>
<h4>Service Part</h4>
In the service part user have server services divided into 4 categories -
<br>
1.Women
<br>
2.Man
<br>
3.Groom
<br>
4.Bridal
<br>
Under the Categories there are many services like - Hair Cur, Makeup ,Spa, etc.
<br>
User can chosse either one or many services at a time 
<h4>Appintment Part</h4>
In the appointment section user can book there appointment by filling appointment form that include - Name , Email, Phone , Time Slot and Date
other then this user can add or remove the selected services and there is live amount calculator which calculate the price .
<br>
The whole Appointment Part is divied into 2 part that on the left side there are user information and on the right side the services and total amount and then at below there is a Confir booking Button 
<br>
After Confir Booking there is a Booking Summary page where user can see there Name, Email, Phone ,Services they choose, and the total Price.
Over there 2 buttons are present -
<br>
1.Edit - To edit any information or either add or remove some service
<br>
2.Process - To process into Payment part
